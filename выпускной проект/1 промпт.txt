Сайт с ИИ-консультантом для онлайн-школы

Цель проекта:

Создание веб-сайта для онлайн-школы с интеграцией ИИ-консультанта (в формате виджета), разработанного с помощью платформы Voiceflow. Консультант будет предоставлять информацию о курсах, расписании и других вопросах.

Общая информация о проекте:

Проект включает создание веб-сайта для онлайн-школы и интеграцию ИИ-консультанта через виджет, встроенный в сайт.

Область применения:

- Описание проблемы:
Пользователи нуждаются в оперативной поддержке при выборе курсов, информации о расписании и записи.

- Пользователи системы:
Потенциальные и текущие студенты, администраторы.

Функциональные требования:

- Веб-сайт:
- Информация о курсах и расписании.
- Раздел отзывов и оценок курсов.
- Остальное наполнение по желанию.

- ИИ-консультант (через Voiceflow):
- Ответы на часто задаваемые вопросы (цена, расписание, формат обучения).
- Рекомендации по выбору курса на основе интересов пользователя.
- Сбор заявок на курсы.
- Возможность интеграции с базами знаний.

Общая архитектура системы:

- Инструменты:
- Платформа Voiceflow для создания и кастомизации ИИ-консультанта.
- Конструктор веб-сайтов Creatium для создания и интеграции виджета.
- API платформы OpenAI для подключения ассистента.

- Подсистемы:
- Интерактивный виджет ИИ-консультанта.
- Каталог курсов.

Методы и стратегии тестирования:

- Настройка и тестирование в Voiceflow:
- Проверка функциональности блоков.
- Убедиться, что ассистент корректно отвечает на запросы.

- Интеграция с веб-сайтом Creatium:
- Вставка HTML-кода виджета в Creatium.
- Проверка отображения виджета на сайте.

- Тестирование ассистента на реальных запросах:
- Проверка корректности ответов из базы знаний.
- Оценка взаимодействия в режиме реального времени.

Этапы выполнения:

1. Подготовка ассистента:
- Создание аккаунта в Voiceflow.
- Настройка ассистента: импорт шаблона, замена API-ключа и Assistant ID.

2. Разработка сайта:
- Использование Creatium для создания структуры сайта.
- Вставка виджета ассистента.

3. Тестирование и отладка:
- Проверка работоспособности ассистента в браузере.
- Контрольное тестирование функциональности виджета.
